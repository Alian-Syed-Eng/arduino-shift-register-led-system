# Modular LED Control System Using Arduino & Shift Registers

## Overview
This project demonstrates a modular embedded system utilizing an Arduino microcontroller and SN74HC595 shift registers to expand digital outputs efficiently.

The system controls multiple LEDs through serial-to-parallel communication, enabling scalable output control while minimizing microcontroller I/O usage.

## Features
- Serial-to-parallel expansion using SN74HC595 shift registers
- LED pattern and diagnostic control
- Modular and scalable hardware design
- Embedded C/C++ firmware (Arduino)

## Hardware Used
- Arduino Uno
- SN74HC595 Shift Registers
- LEDs and 220Ohm resistors
- Breadboard and jumper wires

## Skills Demonstrated
- Embedded Systems
- Digital Logic
- Arduino (C/C++)
- Hardware Debugging
- Bitwise Operations
